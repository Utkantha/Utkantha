## 🚀 Featured Projects

### 🔋 [Electricity Consumption Prediction](https://github.com/Utkantha/Electricity_consumption_prediction)
> Hybrid deep learning model for household electricity forecasting using a **CNN-BiGRU-Self Attention** architecture.

- Applied **MIC (Maximal Information Coefficient)** for feature selection on the UCI household power dataset
- Built and benchmarked against **SVM** and **Linear Regression** baselines — achieved superior R², lower RMSE and MAE
- Deployed via **Flask REST API** with an admin-authenticated web interface for real-time predictions
- **Stack:** Python · TensorFlow/Keras · scikit-learn · NumPy · Pandas · Matplotlib · Flask

---

### 🛒 [Yamazone — E-Commerce Demo](https://yamazone.vercel.app/) &nbsp;|&nbsp; [Source](https://github.com/Utkantha/yamazone)
